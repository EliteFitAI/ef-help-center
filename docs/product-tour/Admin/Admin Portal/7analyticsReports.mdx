# Analytics / Reports

The Analytics section in the EliteFit.AI Admin Portal provides insights into user performance and activity. Follow the steps below to access and interpret key reports.

---

## To View the Overall Report:

1. Navigate to the **Analytics** section and click on **"Report"** to view available report options.
2. To view workout-related data, click on **"Workout"** – this includes workout history, rep counts, accuracy, and workout dates.
3. To see individual user/patient details, click on **"Personal"** – this includes basic user information.

## Sample Report

The Physiotherapy Program Report offers a summarized overview of a patient’s treatment plan and performance analytics.

### Key Sections:

- **Patient Information**: Basic details such as name, age, height, weight, and assigned practitioner.
- **Prescribed Workouts**: Displays the treatment program name and a list of assigned exercises.
- **Feedback Summary**: Practitioner feedback and total counts reflecting patient performance and form quality.
- **Workout Log**: A table showing daily performance metrics, including sets, reps, and time spent.
- **Exercise Report Panel**: Includes session frequency, duration, and number of sets per day.
- **Performance Summary**: Highlights average session duration, accuracy rate, and completion status.
- **Movement Analysis Chart**: Visual representation of the patient’s movement performance across sessions.

This report helps practitioners track progress and adjust treatment plans accordingly.
