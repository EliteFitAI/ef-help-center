# Admin Portal – Adding Patients

Admins can easily add new patients to the system through the User Management section in the Admin Portal.

---

## End User panel

After logging in with the admin credentials, admins will be redirected to the Admin Portal.

1. To add new patients, click **Users**, under **User Management**.
2. Click to invite the user as either an **Admin** or an **End User**, based on the intended role.
3. Click the **“Invite user”** button. A pop-up will appear.
