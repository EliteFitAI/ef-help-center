# Viewing progress

This guide shows you how to track your overall progress for each exercise using the reporting feature.

---

## Workout progress screen

To view your overall progress for each exercise:

1. **Click on “Progress”** in the left panel.  
2. **Click on a thumbnail** of an exercise.  
3. **View the graph** generated for the report.
