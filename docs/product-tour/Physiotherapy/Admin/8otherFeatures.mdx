This section allows admins to customize the appearance and branding of their microsite.

## Steps to customize your microsite:

<div style={{ display: 'flex', alignItems: 'center', gap: '1rem', flexWrap: 'wrap' }}>

<div style={{ flex: 1, minWidth: '300px' }}>

1. **Click "Settings"** from the admin side panel.
2. **Set a custom domain name** for your microsite.
3. **Upload your logo** to personalize the microsite, sign-in, and sign-up pages.
4. **Choose a color theme** that reflects your brand identity.
5. **Preview your changes** in both computer and mobile views before finalizing.
6. **Click "Save Changes"** to apply and save all customizations.

These options help ensure the microsite aligns with your organization's visual identity.

</div>

<ol>
    <li>Write the name of the achievement (example: <em>10-Day Workout Streak</em>).</li>
    <li>Write a short description explaining the achievement (example: <em>Awarded for completing a workout every day for 10 days</em>).</li>
    <li>Upload an achievement badge image (recommended size: <strong>1920 x 1080 px</strong>, <strong>16:9</strong> ratio).</li>
    <li>Select the <strong>start date</strong> from which the achievement tracking should begin.</li>
    <li>Select the <strong>end date</strong> after which the achievement tracking should stop.</li>
    <li>Choose the specific weekdays this achievement should apply to (example: <strong>M, W, F</strong> for Monday, Wednesday, Friday).</li>
    <li>Select the <strong>evaluation method</strong> (<em>Per Workout</em> or <em>All Workouts</em>).</li>
    <li>Enter the <strong>number of points</strong> to be awarded for achieving this badge.</li>
    <li>Set the <strong>condition(s)</strong> under which this achievement will be granted (example: <em>workout duration &gt; 30 mins</em>).</li>
    <li>Toggle the <strong>"Activate"</strong> option to enable or disable tracking for this achievement.</li>
    <li>Toggle <strong>"Can Achieve only once"</strong> if users should only earn this achievement one time.</li>
    <li>Click <strong>"Submit"</strong> to save and create the achievement.</li>
  </ol>

## Send notifications

<div style={{
  display: 'flex',
  flexDirection: ['column', 'row'],
  gap: '2rem',
  margin: '2rem 0'
}} class="discription">
  {/* Left side - Content */}
  <div style={{ flex: 1 }}>
    
    <p style={{margin: 0}}>In the “Send Notification” page users will see the following data fields.</p>
    
    <ol style={{ paddingLeft: "10px" }}>
      <li>
        <strong>Title:</strong> Here the user will enter the notification title, which can be a single word or multiple words.
      </li>
      <li>
        <strong>Message:</strong> This field will have the notification description or the main message of the notification
      </li>
      <li>
        <strong>Icon:</strong> This is where the admins will add an image or GIF for the notification icon.
      </li>
      <li>
        <strong>Notification type:</strong> Here the admin can select from two options, “In-app notification” which are notification that appear only in the application, and second “Push notification” which appear on the device notification panel as long as the user is logged in.
      </li>
      <li>
        <strong>Notification Duration:</strong> Here the user can set how long will the notification float on the user screen
      </li>
      <li>
        <strong>Send Options - Schedule to send:</strong> The admin can select a date and time for when he wants to schedule the notification for
      </li>
      <li>
        <strong>Send Immediately:</strong> Selecting this option sends the notification as soon as the submit button is clicked
      </li>
      <li>
        <strong>Receivers:</strong> The admin can select between individual or multiple users to send a notification to
      </li>
      <li>
        <strong>Send to all:</strong> Selecting this option sends the notification to every user, including the admin.
      </li>
    </ol>
  </div>
