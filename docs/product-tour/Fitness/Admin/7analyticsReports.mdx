# Analytics

The *Analytics* section allows you to generate, filter, and export reports for better insights into user activities and team performance.
Accessing reports
1. Go to the *Analytics* section and click on Reports.
1. A dropdown will appear with two options: *Workouts* and *Personal*.
1. Select the type of report you want to generate.

## Exporting Detailed Reports

1. On the right corner, click the green button labeled **Export Detailed Report**.

![Treatment plan page](/img/WorkoutRep3.webp)

2. When you click this button, an **Export Detailed Report** box will appear.
3. This option allows you to directly export and download detailed workout reports.
4. In the Export Detailed Report box:

   **Workout Dates:**
   - Choose **From Inception to Today**, or
   - Select **Specific Dates** for which you want to export the report.

   **Include or Exclude:**
   - Demographics Data
   - Assessments Data
   - User Feedback

![Treatment plan page](/img/WorkoutRep4.webp)

5. Click **Export**, and the report will be downloaded automatically.

![Treatment plan page](/img/WorkoutRep5.webp)

---

## Scheduling Reports

1. Next to the **Export Detailed Report** button, click the dropdown arrow.
2. You will see two options: **Schedule** and **History**.

![Treatment plan page](/img/WorkoutRep10.webp)

### Schedule

3. Under **Schedule**, you can:
   - Set the frequency: **Daily**, **Weekly**, or **Monthly**.
   - Select the **Delivery Time**.
   - Enter the **Email Recipient** where the scheduled report should be sent.
   - Choose to export data **From Inception to Today** or for a **Specific Date Range**.

![Treatment plan page](/img/WorkoutRep6.webp)

   - Include or exclude:
     - Demographics data
     - Assessments data
     - User feedback

![Treatment plan page](/img/WorkoutRep7.webp)

4. Click **Schedule** to confirm.

![Treatment plan page](/img/WorkoutRep8.webp)

### History

5. Under **History**, you can view all the reports you have scheduled previously, along with their details and status.

![Treatment plan page](/img/WorkoutRep9.webp)
