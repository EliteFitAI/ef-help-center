
# Notifications

The Notification section allows the admin to create and send important updates to users through in-app or push notifications.

---

## Access Notification Section

1. Go to the left panel.
2. Under Utilities, click on Notification.
3. The Notification page will open.

![EliteFit.AI Library](/img/PhysioNotification1.webp)

--- 

## Send a New Notification

1. On the top right corner, click on the Send Notification button.
2. The Notification Form will open up.

![EliteFit.AI Library](/img/PhysioNotification1.webp)

---

## Fill in Notification Details

Inside the form, the admin can add:

1. Title of the notification
2. Description or message content
3. Icon or Image (optional)

![EliteFit.AI Library](/img/PhysioNotification2.webp)

---

## Set Notification Type

Under Notification Settings, you can choose:

1. In-App Notification
2. Push Notification

![EliteFit.AI Library](/img/PhysioNotification4.webp)

---

## Set Notification Duration

1. Enter the notification duration in seconds using the duration input field.
2. Set the exact number of seconds for how long the notification should stay visible.

![EliteFit.AI Library](/img/PhysioNotification3.webp)

---

## Scheduling Options

The admin can choose how and when to send the notification:

1. Send Immediately – Sends the notification right away.
2. Schedule for Later – Select a specific date and time from the calendar.

![EliteFit.AI Library](/img/PhysioNotification3.webp)

---

## Select Recipients

You can decide who will receive the notification:

1. Send to a specific user – Enter the username in the receiver field.
2. Send to all users – Enable the option Send to All.

![EliteFit.AI Library](/img/PhysioNotification5.webp)

---

## Submit Notification

1. After filling in all details, click on Submit.
2. The notification will be sent based on the chosen settings.

![EliteFit.AI Library](/img/PhysioNotification6.webp)
