# Achievements

The Achievements section allows admins to create badges that users can earn based on their workout performance or program completion.

---

## Creating a new achievement

1. Go to the Achievements section in the left panel.
2. The Achievements page will open.
3. In the top right corner, click the green Create Achievement button.

![EliteFit.AI Library](/img/FitnessAchievement1.webp)

4. The achievement form will open.
5. Enter the details such as Achievement Name, Description, and Badge Image.

![EliteFit.AI Library](/img/FitnessAchievement2.webp)

---

## Configure achievement duration

1. In the Achievement Configuration section, select the Start Date and End Date for the achievement badge.
2. You can also select specific weekdays.
3. If you do not select any weekday, all days will be included by default.

![EliteFit.AI Library](/img/FitnessAchievement5.webp)

---

## Set the evaluation method

Choose one evaluation method:

![EliteFit.AI Library](/img/FitnessAchievement3.webp)

1. Per Workout

![EliteFit.AI Library](/img/FitnessAchievement5.webp)

2. All Workouts

![EliteFit.AI Library](/img/FitnessAchievement4.webp)

3. Per Program
    -  If you select Per Program, an additional tab will appear.
    -  Select the program from the list of existing programs.

![EliteFit.AI Library](/img/FitnessAchievement6.webp)

---

## Add conditions

1. For Per Workout or All Workouts, the admin can add one or more conditions.
2. Conditions can be selected based on the requirement of the badge.

![EliteFit.AI Library](/img/FitnessAchievement7.webp)

![EliteFit.AI Library](/img/FitnessAchievement8.webp)

---

## Assign points (optional)

1. The admin can assign points to the badge if needed.

![EliteFit.AI Library](/img/FitnessAchievement10.webp)

---

## Set achievement frequency and activation

1. Select whether the badge will be active or inactive.
2. Select whether the badge can be achieved only once or multiple times.

![EliteFit.AI Library](/img/FitnessAchievement10.webp)

<Admonition type="info" title="Per Program">
  Achievements configured as Per Program can only be earned once, as each program can be completed by a user only a single time.
</Admonition>

![EliteFit.AI Library](/img/FitnessAchievement11.webp)

---

## Submit the achievement
1. Click on Submit.
2. The achievement badge will be created successfully.

![EliteFit.AI Library](/img/FitnessAchievement9.webp)
